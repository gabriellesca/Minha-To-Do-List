## 🎯 Sobre o projeto

Projeto desenvolvido para praticar manipulação dinâmica do DOM e persistência de dados no front-end. O diferencial deste projeto é que as tarefas **continuam salvas mesmo após fechar o navegador**, usando o LocalStorage do próprio navegador como banco de dados.

---

## ✨ Funcionalidades

- [x] Adicionar tarefas com clique ou tecla **Enter**
- [x] Marcar/desmarcar tarefa como concluída
- [x] Remover tarefa individualmente
- [x] Persistência automática com **LocalStorage**
- [x] Contador de tarefas em tempo real
- [x] Mensagem de lista vazia
- [x] Animações de entrada com CSS puro

## 🧠 Conceitos aplicados

| Conceito | Descrição |
|---|---|
| `createElement` | Criação dinâmica de elementos no DOM |
| `appendChild` | Inserção de elementos na lista |
| `classList.toggle` | Alternar classe de concluída no clique |
| `localStorage` | Persistência dos dados no navegador |
| `JSON.stringify` / `JSON.parse` | Serialização e leitura dos dados salvos |
| `keydown` + `e.key` | Escuta de teclas específicas (Enter) |
| Parâmetro padrão | `completed = false` em funções ES6 |
| Funções separadas | `createTask`, `saveTasks`, `loadTasks` |
